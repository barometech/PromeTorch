# Источники для документации ISA E2K

Список upstream-источников из которых можно извлекать сведения для
документации. Каждый источник имеет свою лицензию — соблюдай её при
цитировании.

## 1. QEMU-эмулятор Эльбрус

**Репозиторий:** https://github.com/OpenE2K/qemu-e2k
**Лицензия:** GPL-2

Самый ценный публичный источник по ISA E2K. Содержит исполняемое
описание команд, регистров, форматов кодирования.

| Файл | Что внутри |
|------|-----------|
| `target/e2k/cpu.h` | Регистры, флаги, CPUE2KState, MMU |
| `target/e2k/alops.inc` | Таблица всех ALU операций (mnemonics, форматы, channel mask) |
| `target/e2k/opcodes-e2k.h` | Перечисление опкодов |
| `target/e2k/opcodes-e2k-inl.h` | Inline-описания опкодов |
| `target/e2k/translate.c` | Декодирование широкой команды → TCG ops |

Подход: парсить `alops.inc` грепом, генерировать таблицы mnemonics →
формат → каналы. Регистры — извлечь из `cpu.h` блока `enum`.

## 2. Сообщество E2K — awesome-list

**Репозиторий:** https://github.com/e2k-community/awesome-e2k

Свалка всего открытого по Эльбрусу: статьи на Хабре, видеоуроки,
бенчмарки, презентации с конференций, прошивки, тулчейны.

## 3. Патчи / порты GNU toolchain

**Репозиторий:** https://github.com/ilyakurdyukov/e2k-ports

Патчи к binutils, glibc, gcc-elbrus. Полезно смотреть как реализованы
архитектурно-зависимые места — кодирование инструкций, ABI, calling
conventions.

## 4. Официальная документация МЦСТ

**Доступ:** под NDA. Заявка через ОК «Сообщество разработчиков
Эльбрус».

| Документ | О чём |
|----------|-------|
| Elbrus Programming Guide (rev. 2024-12) | Полный референс компилятора LCC |
| Elbrus-2000-opc.pdf | Опкоды (для внутренних разработчиков) |
| elbrus_alc_formats.pdf | Форматы ALC-команд |

⚠ **Цитировать только в виде ссылок** и перефразированных смысловых
фрагментов. **Не копировать целые таблицы / диаграммы** из этих PDF в
открытый репозиторий.
