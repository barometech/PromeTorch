# Руководство по системе команд процессора Эльбрус (E2K)

Открытый проект по созданию **полноценной документации ISA Эльбрус**
сопоставимой по качеству с Intel SDM / AMD APM / ARM ARM. Назначение:
системные программисты, разработчики компиляторов, embedded-инженеры,
исследователи VLIW-архитектур.

## Зачем

У сообщества нет нормальной публичной документации на систему команд
E2K. Официальный МЦСТ-овский «Elbrus Programming Guide» закрыт за NDA,
доступен только под подпись. Открытые источники разбросаны:

- Заголовки QEMU-эмулятора процессора Эльбрус
- Свалка `awesome-e2k`
- Патчи разработчиков под `e2k-ports`
- Отдельные доклады, статьи на Хабре, презентации

Этот документ собирает всё воедино и приводит к единообразному формату.

## Как контрибьютить

1. Выбери раздел из ROADMAP с пометкой `TODO`.
2. Открой issue с пометкой `[isa-doc]` и названием раздела.
3. Создай файл `chapter_NN_<slug>.md` в этой директории.
4. Используй [SOURCES.md](SOURCES.md) как первоисточник — извлекай
   сведения оттуда, не копируй цельные блоки текста из закрытых
   PDF'ов МЦСТ.
5. PR с указанием issue.

### Требования к стилю

- **Язык:** русский. Английские термины оставляй в скобках при первом
  упоминании: «широкая команда (wide instruction)».
- **Точность:** каждый mnemonic / формат / register должен быть
  подкреплён ссылкой на источник (commit hash QEMU, страница MCST PDF,
  и т.п.).
- **Примеры:** каждый mnemonic + 1 короткий ассемблерный пример.
- **Перекрёстные ссылки:** через relative markdown links
  `[Канал 0](#канал-0)`, не абсолютные URL.
